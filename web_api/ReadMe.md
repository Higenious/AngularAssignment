 ## web_api

### Technology Required
 + Node js
 + Mongo db
 + Express 

### How to Start
 - Download zip or Clone project.
 - Navigate to folder.
 - npm install 
 - npm start  



```sh
$ http://localhost:5000/api/v1/user/register
$ http://localhost:5000/api/v1/user/getall
$ http://localhost:5000/api/v1/user/edit
$ http://localhost:5000/api/v1/user/delete
$ http://localhost:5000

```
